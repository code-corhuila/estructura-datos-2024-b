¿Qué es una Función?

Una función es un bloque de código diseñado para realizar una tarea específica. Las funciones en JavaScript permiten organizar y reutilizar el código, lo que facilita la creación de programas eficientes y modulares.

Tipos de Funciones

  • Funciones Declarativas: Son funciones definidas con la palabra clave function, seguidas del nombre de la función y un conjunto de paréntesis.
  • Funciones Anónimas: Son funciones sin nombre, normalmente asignadas a una variable. Son útiles en funciones de retorno o en funciones pasadas como argumentos.
  • Funciones Flecha (Arrow Functions): Introducidas en ES6, tienen una sintaxis más compacta y se definen con el símbolo =>. No poseen su propio contexto this, lo cual las hace ideales para funciones de callback.

¿Para Qué Sirve una Función?

Las funciones son utilizadas para dividir el código en secciones manejables y reutilizables, lo cual facilita la lectura y el mantenimiento del programa. Al encapsular una tarea dentro de una función, es posible invocar esa función varias veces en distintos lugares del código, lo que ahorra tiempo y minimiza errores.

¿Cómo se Define una Función?

En JavaScript, existen varias formas de definir funciones. Aquí tienes algunos ejemplos:

Ejemplo Uno: Función Declarativa

function nombreFuncion(parametro1, parametro2) {
    // Código de la función
}

Ejemplo:

function saludar(nombre) {
    return "Hola, " + nombre;
}

Ejemplo Dos: Función Anónima

const miFuncion = function(parametro) {
    // Código de la función
};

Ejemplo:

const multiplicar = function(a, b) {
    return a * b;
};

Ejemplo Tres: Función Flecha

const nombreFuncion = (parametro1, parametro2) => {
    // Código de la función
};

Ejemplo:

const sumar = (a, b) => a + b;
